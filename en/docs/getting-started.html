Quick Start

This is a regular page that includes the basics of VuePress.

Pages

You can add Markdown files to your VuePress directory, and each Markdown file will be converted into a page on your site.

For more details, please refer to Routing.

Content

Each Markdown file will be rendered as HTML and then transformed into a single-file Vue component (SFC).

VuePress supports basic Markdown syntax and some extensions, and you can also use Vue features within Markdown.

Configuration

VuePress uses a .vuepress/config.js (or .ts) file as the site configuration, which you can use to configure your site.

For client-side configuration, you can create a .vuepress/client.js (or .ts) file.

Additionally, you can add configuration for each page using frontmatter.

Layout and Customization

Here are common configurations for controlling the layout of the @vuepress/theme-default theme:

  • Navbar
  • Sidebar

Refer to the Default Theme Documentation for complete reference information.

You can add extra styles by creating a .vuepress/styles/index.scss file.
